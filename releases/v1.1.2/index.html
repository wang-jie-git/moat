    Bug 修复¶

    fix(cli): 修复 moat check --diff KeyError: 'callers'¶

    问题: moat check --diff 模式崩溃，出现 KeyError: 'callers'

    根因: analyze_impacts() 返回的字典缺少 callers 字段，只有 direct_callers 和 indirect_callers

    修复: 在 moat/ast/builder.py:284-294 添加 callers 字段

    # 修复前
if direct_callers or indirect_callers:
    impacts.append({
        "change": change,
        "direct_callers": direct_callers,
        "indirect_callers": indirect_callers,
        ...
    })

# 修复后
if direct_callers or indirect_callers:
    all_callers = direct_callers + indirect_callers
    impacts.append({
        "change": change,
        "callers": all_callers,  # ✅ 新增：兼容 cli.py 访问方式
        "direct_callers": direct_callers,
        "indirect_callers": indirect_callers,
        ...
    })

    验证: 

    $ moat check --diff
...
💡 影响域分析:
   📍 moat/ast/builder.py::build_skeleton
      影响 1 个调用方:
        - {'caller': 'moat/cli.py::cmd_check', 'confidence': 1.0}
      风险等级: low
✅ 变更风险较低，但仍建议运行完整检查

    影响范围: 仅影响 moat check --diff 模式，不影响 --quick 和 --full

    向后兼容: ✅ 保留 direct_callers / indirect_callers，新增 callers

    测试: tests/test_diff_bugfix.py
