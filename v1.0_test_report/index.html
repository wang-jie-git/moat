    发现的 Bug¶

    Bug #1: FullCheck.init 参数错误¶

    错误信息: 

    TypeError: Check.__init__() takes from 2 to 3 positional arguments but 4 were given

    位置: moat/checks/quick_check.py:241

    原因: FullCheck 调用了 super().__init__("FullCheck", project_root, config)，但基类 Check.__init__ 只接受 (self, project_root, config) 两个参数。

    修复: 移除多余的 "FullCheck" 参数，改为 self.name = "FullCheck"

    状态: ✅ 已修复
